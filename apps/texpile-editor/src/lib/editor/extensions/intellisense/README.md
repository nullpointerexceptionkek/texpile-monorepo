Most of what's in this folder is heavily inspired by the [LaTeX Workshop](https://github.com/James-Yu/LaTeX-Workshop) VS Code extension's IntelliSense (completion, hover, folding, go-to-definition), reimplemented for CodeMirror. Data comes from unified-latex's curated CTAN signature DB plus LaTeX Workshop's own default macro/environment set (vendored in `data/lwMacros.ts` and `completion/environments.ts` extras) instead of its ~250-package CTAN scrape.
